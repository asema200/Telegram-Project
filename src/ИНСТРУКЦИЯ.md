# Инструкция по запуску системы

## Шаг 1. Получение токена Telegram бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду: `/newbot`
3. Введите имя бота (например: "Career Assistant Bot")
4. Введите username бота (должен заканчиваться на `bot`, например: `career_assistant_bot`)
5. Скопируйте полученный токен (формат: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

---

## Шаг 2. Настройка конфигурации

Создайте файл `.env` в корневой директории проекта со следующим содержимым:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

**Важно:** Замените `ваш_токен_от_BotFather` на токен, полученный на Шаге 1.

---

## Шаг 3. Установка зависимостей

### Windows:

Откройте командную строку (cmd) в директории проекта:

```cmd
python -m venv venv
venv\Scripts\pip install -r requirements.txt
```

### macOS / Linux:

Откройте терминал в директории проекта:

```bash
python3 -m venv venv
venv/bin/pip install -r requirements.txt
```

---

## Шаг 4. Запуск системы

### Автоматический запуск (рекомендуется):

**Windows:**
```cmd
start.bat
```
(или дважды кликните на файл)

**macOS / Linux:**
```bash
chmod +x start.sh
./start.sh
```

### Ручной запуск:

**Windows:**
```cmd
cd src
..\venv\Scripts\python main_advanced.py
```

**macOS / Linux:**
```bash
cd src
../venv/bin/python3 main_advanced.py
```

---

## Шаг 5. Проверка работы

1. Откройте Telegram
2. Найдите созданного бота по username
3. Нажмите кнопку **START** или отправьте команду `/start`
4. Если бот ответил приветственным сообщением - система работает корректно

---

## Использование системы

### Основные команды:

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота и отображение главного меню |
| `/help` | Справочная информация |
| `/settings` | Настройка параметров поиска |
| `/compare` | Сравнение нескольких профессий |
| `/quiz` | Профориентационное тестирование |

### Примеры использования:

**Поиск вакансии:**
```
программист
```
Результат: статистика по зарплатам и график распределения

**Сравнение профессий:**
```
/compare программист, дизайнер, аналитик данных
```
Результат: сравнительный график и таблица

**Профориентация:**
```
/quiz
```
Результат: ответьте на 7 вопросов, получите рекомендации

## Решение типичных проблем

### Проблема: "No module named 'telebot'"

**Решение:**
```bash
pip install -r requirements.txt
```

### Проблема: "Токен не найден"

**Решение:**
1. Убедитесь что файл `.env` создан
2. Проверьте формат: `BOT_TOKEN=ваш_токен`
3. Файл должен находиться в корневой директории

### Проблема: Error 409 "Conflict"

**Причина:** Запущено несколько экземпляров бота одновременно

**Решение:**
- Windows: завершите все окна с запущенным ботом
- macOS/Linux: выполните `pkill -f main_advanced.py`
- Запустите только одну копию

### Проблема: Бот не отвечает

**Проверьте:**
1. Корректность токена в `.env`
2. Наличие интернет-соединения
3. Запущен ли процесс бота
4. Отсутствие ошибок в консоли
